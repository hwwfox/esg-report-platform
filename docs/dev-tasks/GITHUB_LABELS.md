# Suggested GitHub Issue Labels

建议在代码仓库中创建以下 labels，方便 Codex/开发者按模块筛选任务。

| Label | 用途 |
|---|---|
| area:auth | 认证与权限 |
| area:tenant | 多租户 |
| area:project | 企业与项目 |
| area:standard-library | 标准库 |
| area:gics-peer | GICS与同行 |
| area:file | 文件与对象存储 |
| area:ai | AI Agent与模型调用 |
| area:worker | 异步任务 |
| area:recommendation | 推荐算法/推荐流程 |
| area:collection | 部门采集与审核 |
| area:knowledge | 知识库 |
| area:report-writing | 报告写作 |
| area:citation | 来源引用 |
| area:review | AI校对 |
| area:export | 导出 |
| area:security | 安全 |
| type:feature | 新功能 |
| type:bug | 缺陷 |
| type:chore | 工程配置 |
| type:docs | 文档 |
| priority:p0 | 必须完成 |
| priority:p1 | 重要 |
| priority:p2 | 可延后 |
